    bartificer. ca

    This namespace provides a JavaScript API for creating Cellular Automata within HTML documents.

    This API uses jQuery to interact with the DOM, so jQuery must be imported into the HTML document before this API.

    See:
    • https://jquery.com/
    • https://en.wikipedia.org/wiki/Cellular_automaton
